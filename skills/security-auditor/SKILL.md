# security-auditor

## Purpose
Use this skill when reviewing forms, scripts, links, schema, or trust-sensitive content.

## What to do
- Check for insecure external links and broken assumptions.
- Review forms for safe handling and clear user intent.
- Ensure no secrets, tokens, or private data are exposed.
- Check structured data for correctness and consistency.
- Look for obvious XSS, injection, or trust issues in dynamic content.

## Good output
- Concrete risk findings
- Severity ordering
- Practical remediation notes
