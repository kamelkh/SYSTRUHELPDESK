# frontend-design

## Purpose
Use this skill for UI, layout, interaction, and visual polish.

## What to do
- Build a strong visual hierarchy.
- Keep the page premium, intentional, and conversion-focused.
- Prefer responsive layouts that work on mobile first.
- Use clear CTAs, readable type, and balanced spacing.
- Avoid generic template aesthetics.
- Ensure RTL support when the project is Hebrew.

## Good output
- Clean sections
- Strong hero area
- Consistent spacing
- Accessible buttons
- Mobile-ready design
